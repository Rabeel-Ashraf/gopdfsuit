Usage Examples

Generate PDF with cURL 
curl -X POST "http://localhost:8080/api/v1/generate/template-pdf" \
  -H "Content-Type: application/json" \
  -d '{"config":{"page":"A4","pageAlignment":1},"title":{"props":"font1:20:100:center:0:0:2:0","text":"Sample Document"}}' \
  --output document.pdf
Generate PDF with Python 
import requests

url = "http://localhost:8080/api/v1/generate/template-pdf"
template = {
    "config": {"page": "A4", "pageAlignment": 1},
    "title": {"props": "font1:20:100:center:0:0:2:0", "text": "Sample Document"}
}

response = requests.post(url, json=template)
with open("document.pdf", "wb") as f:
    f.write(response.content)
